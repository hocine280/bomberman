Joue la flèche, vérifie si la flèche une fois déplacé est sur la même case que le \mbox{\hyperlink{class_bomberman}{Bomberman}}, si c\textquotesingle{}est le cas inflige des dégâts au \mbox{\hyperlink{class_bomberman}{Bomberman}} sinon vérifie si la flèche n\textquotesingle{}est pas sortie de la map ou si la case sur laquelle elle souhaite se déplacer est libre. Si la flèche serait encore dans la map et que la case est libre alors déplace la flèche.
